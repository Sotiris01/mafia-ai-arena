# E13 — Strange Illness (Παράξενη Ασθένεια)
---

| Property         | Value                                                         |
| ---------------- | ------------------------------------------------------------- |
| **ID**           | `E13`                                                         |
| **Type**         | Night Echo Event                                              |
| **Timing**       | 💬 Mid-Day                                                    |
| **Probability**  | `0.20`                                                        |
| **Linked Roles** | [[Zombie]]                                                     |
| **Trigger**      | Ο Zombie **μόλυνε** κάποιον χθες βράδυ (πέρα από το official announcement) |
| **Suspicion Weight** | +0.20                                                     |
| **Red Herring**  | 30% chance να στοχεύσει λάθος παίκτη                          |

---

## Message

> _"Ο Player X δεν φαίνεται καλά... μήπως κάτι τον επηρεάζει;"_

---

## Τι Αποκαλύπτει

Ενισχύει ότι ο Player X **μολύνθηκε** — cross-reference με zombie announcement.

## Τι Κρύβει

Πιθανό red herring αν ο Player X δεν είναι zombie.

> **Σημαντικό:** Αυτό εμφανίζεται **μόνο αν ο Zombie ρόλος υπάρχει**, αλλά μπορεί να στοχεύσει **οποιονδήποτε** παίκτη (red herring 30% ή actual zombie victim 70%).
